# Gebakoo E-commerce web app

# Get started
* First clone Gebakoo_MVP repository <br /><br />
  git clone https://github.com/BirtukanK/Gebakoo_MVP <br />
  cd Gebakoo_MVP <br />

* install dependencies <br />
  npm install <br />

* Run application with this command <br />
  npm start <br />
